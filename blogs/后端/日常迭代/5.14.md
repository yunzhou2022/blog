# MyBatis 教学

## 介绍

MyBatis 是一款优秀的持久层框架，它支持自定义 SQL、存储过程以及高级映射。MyBatis 免除了几乎所有的 JDBC 代码以及设置参数和获取结果集的工作。MyBatis 可以通过简单的 XML 或注解来配置和映射原始类型、接口和 Java POJO（Plain Old Java Objects，普通老式 Java 对象）为数据库中的记录。

与 Hibernate 等全自动 ORM 框架相比，MyBatis 是半自动的，SQL 由开发者自己编写，灵活性更强，更容易做性能优化。

## 安装配置启用

### Maven 依赖

```xml
<dependency>
    <groupId>org.mybatis.spring.boot</groupId>
    <artifactId>mybatis-spring-boot-starter</artifactId>
    <version>3.0.3</version>
</dependency>
<dependency>
    <groupId>com.mysql</groupId>
    <artifactId>mysql-connector-j</artifactId>
    <scope>runtime</scope>
</dependency>
```

### application.yml 配置

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/demo?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
    username: root
    password: your_password
    driver-class-name: com.mysql.cj.jdbc.Driver

mybatis:
  mapper-locations: classpath:mapper/*.xml   # XML 文件位置
  type-aliases-package: com.example.entity   # 实体类包路径
  configuration:
    map-underscore-to-camel-case: true        # 下划线转驼峰
    log-impl: org.apache.ibatis.logging.stdout.StdOutImpl  # 开发时打印 SQL
```

### 启用 Mapper 扫描

在启动类上加 `@MapperScan`：

```java
@SpringBootApplication
@MapperScan("com.example.mapper")
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}
```

## 跑通第一条 SQL

### 建表

```sql
CREATE TABLE user (
    id      BIGINT PRIMARY KEY AUTO_INCREMENT,
    name    VARCHAR(64)  NOT NULL,
    email   VARCHAR(128) NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

### 实体类

```java
public class User {
    private Long id;
    private String name;
    private String email;
    private LocalDateTime createdAt;
    // getter / setter 省略
}
```

### Mapper 接口

```java
@Mapper
public interface UserMapper {
    User selectById(Long id);
}
```

### Mapper XML

`resources/mapper/UserMapper.xml`：

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE mapper PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-mapper.dtd">
<mapper namespace="com.example.mapper.UserMapper">
    <select id="selectById" resultType="User">
        SELECT * FROM user WHERE id = #{id}
    </select>
</mapper>
```

### 调用

```java
@Service
public class UserService {
    @Autowired
    private UserMapper userMapper;

    public User getUser(Long id) {
        return userMapper.selectById(id);
    }
}
```

## 单表 CRUD

### Mapper 接口定义

```java
@Mapper
public interface UserMapper {
    User selectById(Long id);
    List<User> selectAll();
    int insert(User user);
    int updateById(User user);
    int deleteById(Long id);
}
```

### XML 实现

```xml
<mapper namespace="com.example.mapper.UserMapper">

    <!-- 通用字段，避免重复写 SELECT * -->
    <sql id="Base_Column_List">
        id, name, email, created_at
    </sql>

    <select id="selectById" resultType="User">
        SELECT <include refid="Base_Column_List"/>
        FROM user WHERE id = #{id}
    </select>

    <select id="selectAll" resultType="User">
        SELECT <include refid="Base_Column_List"/>
        FROM user
    </select>

    <!-- useGeneratedKeys 回填自增主键 -->
    <insert id="insert" useGeneratedKeys="true" keyProperty="id">
        INSERT INTO user (name, email)
        VALUES (#{name}, #{email})
    </insert>

    <update id="updateById">
        UPDATE user
        SET name  = #{name},
            email = #{email}
        WHERE id = #{id}
    </update>

    <delete id="deleteById">
        DELETE FROM user WHERE id = #{id}
    </delete>

</mapper>
```

### 动态 SQL

条件查询常用 `<if>` + `<where>`，避免拼接空条件：

```xml
<select id="selectByCondition" resultType="User">
    SELECT <include refid="Base_Column_List"/>
    FROM user
    <where>
        <if test="name != null and name != ''">
            AND name LIKE CONCAT('%', #{name}, '%')
        </if>
        <if test="email != null and email != ''">
            AND email = #{email}
        </if>
    </where>
</select>
```

批量插入使用 `<foreach>`：

```xml
<insert id="batchInsert" useGeneratedKeys="true" keyProperty="id">
    INSERT INTO user (name, email) VALUES
    <foreach collection="list" item="item" separator=",">
        (#{item.name}, #{item.email})
    </foreach>
</insert>
```

## 多表 CRUD

### 一对一：resultMap + association

场景：`order` 表关联 `user` 表。

```xml
<resultMap id="OrderWithUser" type="Order">
    <id     property="id"     column="order_id"/>
    <result property="amount" column="amount"/>
    <association property="user" javaType="User">
        <id     property="id"    column="user_id"/>
        <result property="name"  column="user_name"/>
        <result property="email" column="user_email"/>
    </association>
</resultMap>

<select id="selectOrderWithUser" resultMap="OrderWithUser">
    SELECT o.id       AS order_id,
           o.amount,
           u.id       AS user_id,
           u.name     AS user_name,
           u.email    AS user_email
    FROM `order` o
    JOIN user u ON o.user_id = u.id
    WHERE o.id = #{id}
</select>
```

### 一对多：resultMap + collection

场景：查询用户及其所有订单。

```xml
<resultMap id="UserWithOrders" type="User">
    <id     property="id"    column="id"/>
    <result property="name"  column="name"/>
    <collection property="orders" ofType="Order">
        <id     property="id"     column="order_id"/>
        <result property="amount" column="amount"/>
    </collection>
</resultMap>

<select id="selectUserWithOrders" resultMap="UserWithOrders">
    SELECT u.id,
           u.name,
           o.id     AS order_id,
           o.amount
    FROM user u
    LEFT JOIN `order` o ON o.user_id = u.id
    WHERE u.id = #{id}
</select>
```

> **注意**：多表查询列名冲突时一定要起别名，否则 MyBatis 映射会出错。

## 自动生成 Example 代码

MyBatis Generator（MBG）可以根据表结构自动生成 Entity、Mapper 接口、Mapper XML 以及 Example 类（用于构建复杂查询条件）。

### 配置 generatorConfig.xml

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE generatorConfiguration PUBLIC
        "-//mybatis.org//DTD MyBatis Generator Configuration 1.0//EN"
        "http://mybatis.org/dtd/mybatis-generator-config_1_0.dtd">
<generatorConfiguration>
    <context id="MySQLTables" targetRuntime="MyBatis3">

        <!-- 数据库连接 -->
        <jdbcConnection driverClass="com.mysql.cj.jdbc.Driver"
                        connectionURL="jdbc:mysql://localhost:3306/demo"
                        userId="root"
                        password="your_password"/>

        <!-- 实体类输出 -->
        <javaModelGenerator targetPackage="com.example.entity"
                            targetProject="src/main/java"/>
        <!-- XML 输出 -->
        <sqlMapGenerator targetPackage="mapper"
                         targetProject="src/main/resources"/>
        <!-- Mapper 接口输出 -->
        <javaClientGenerator type="XMLMAPPER"
                             targetPackage="com.example.mapper"
                             targetProject="src/main/java"/>

        <!-- 要生成的表 -->
        <table tableName="user" domainObjectName="User"/>
    </context>
</generatorConfiguration>
```

### 运行生成

```bash
mvn mybatis-generator:generate
```

### 使用 Example 查询

生成后会得到 `UserExample` 类，通过 Criteria 链式构建条件：

```java
UserExample example = new UserExample();
example.setOrderByClause("created_at DESC");
example.createCriteria()
       .andNameLike("%张%")
       .andEmailIsNotNull();

List<User> users = userMapper.selectByExample(example);
```

> **建议**：生成的代码不要手动修改，如需扩展，另建 `UserMapperExt` 继承生成的 Mapper。

## MyBatis Plus

MyBatis Plus（MP）是 MyBatis 的增强工具，在 MyBatis 基础上只做增强不做改变，内置通用 CRUD、条件构造器、代码生成器等功能，能大幅减少样板代码。

### 依赖

```xml
<dependency>
    <groupId>com.baomidou</groupId>
    <artifactId>mybatis-plus-boot-starter</artifactId>
    <version>3.5.7</version>
</dependency>
```

### 实体类注解

```java
@TableName("user")
public class User {
    @TableId(type = IdType.AUTO)
    private Long id;
    private String name;
    private String email;
    @TableField(fill = FieldFill.INSERT)
    private LocalDateTime createdAt;
}
```

### Mapper 继承 BaseMapper

```java
@Mapper
public interface UserMapper extends BaseMapper<User> {
    // 无需写任何方法，CRUD 全部继承自 BaseMapper
}
```

### 常用操作

```java
// 按主键查
userMapper.selectById(1L);

// 插入（自动回填 id）
userMapper.insert(user);

// 条件查询
LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<User>()
        .like(User::getName, "张")
        .orderByDesc(User::getCreatedAt);
List<User> list = userMapper.selectList(wrapper);

// 分页（需注册分页插件）
Page<User> page = new Page<>(1, 10);
userMapper.selectPage(page, wrapper);
// page.getRecords() / page.getTotal()

// 按条件更新
LambdaUpdateWrapper<User> updateWrapper = new LambdaUpdateWrapper<User>()
        .eq(User::getId, 1L)
        .set(User::getName, "新名字");
userMapper.update(null, updateWrapper);

// 软删除（配置 @TableLogic 后自动过滤已删除记录）
userMapper.deleteById(1L);
```

### 注册分页插件

```java
@Configuration
public class MybatisPlusConfig {
    @Bean
    public MybatisPlusInterceptor mybatisPlusInterceptor() {
        MybatisPlusInterceptor interceptor = new MybatisPlusInterceptor();
        interceptor.addInnerInterceptor(new PaginationInnerInterceptor(DbType.MYSQL));
        return interceptor;
    }
}
```

### MP vs 原生 MyBatis

| 场景 | 推荐 |
|------|------|
| 简单单表 CRUD | MyBatis Plus |
| 复杂多表 JOIN / 子查询 | 原生 MyBatis XML |
| 报表类复杂 SQL | 原生 MyBatis XML |
| 快速开发 CRUD 接口 | MyBatis Plus + 代码生成器 |
