# 莎士比亚的智慧-论朋友

- 只上一次当（提前假设所有人都是好人、如果上当、就移除）
- 三思而行，不要想到什么就说什么。说话前要想好再说，不要毛躁
- 对人要有礼貌，保持距离是朋友之间最好的交往方式
- 好朋友、要好好相处，但是不要对每一个范范新知施舍你的交情

# 佩奇的经营管理智慧（google）

- 把产品做成牙刷，客户每天都用上
- 从本质中寻找商业模式（给客户提供有用的内容： 正：google搜索，反：百度搜索）
- 薪尽火传（原公司业务死了，新业务起来了）

# 巴菲特的智慧

- 不做自己不擅长的事（不意味着一直不擅长）
- 不要做空股票
- 不要用杠杆投资（人一生不要有两次富有，富有起来后不要去冒险）
- 投资股票不如投资自己的专业

# 司马迁的智慧——5档

- 因材施教-》以利导之-》教诲-》立规矩-》对立（故善者因之，其次利道之，其次教诲之，其次整齐之，最下者与之争）
- 任何生意、农业、卖油、贩酒等，只要经营的好，做到极致，就能获得巨大的成功。

# 销售大师的智慧

- 销售的本质是把钱收回来（不能有坏账、可以有欠条、但是要能收回来）
- 让顾客把买到的东西用光，卖出去的东西， 要让用户尽快消耗掉，不然没有回购
- 要让用户觉得自己用这个东西有面子（包装好、像苹果一样，不要整出好几个机型）
