# 我们和天才的差距有多大

差距可能不在基因上，而在别的方面：

- 认知
- 见识
- 做事的方法
- 勇气

与其想着智力，不如学学他们做事的方法，这些是我们可以控制的。

# 起跑线和玻璃心

起跑线好不一定会一直好，起跑线差、也不一定会一直差。

起跑线好、可能会导致玻璃心，受不了挫折，从而一蹶不振。

与其训练有爆发力没有耐力的短跑， 不如练就一颗大心脏，即使跌倒也能爬起来。

# 论运气

运气很重要，但是运气不能把控， 不必对自己太苛求，如果我们有足够的耐心、好的方法和持之以恒的努力，或许好运会降临到我们头上，也或许没有来。这种情况下、我们可以同肯尼迪的话来宽慰自己：“问心无愧是我们唯一能得到的报酬。” 因为我们能做的只是“尽人事、听天命”

# 比贫穷更可怕的是什么

- 没有规矩（闯祸）
- 缺爱（不会分享、没朋友）
- 没有见识 （讲道理讲不通）

# 与庄子对话谈谈见识

很多人不愿意接受新的事物，是因为他们满足于自身的成就和环境，或者他们已经觉得自己了不起。

当我们看到眼前、看到周围的时候，做了一件好事，受到一些夸奖，就不免沾沾自喜，喜欢拿来显摆，但是如果把它拿到大时代中、我们做的这一点事情就不算什么了。很多人看似很重要的事，如果跳出自己的认知局限、其实是可有可无的。

并不是什么东西都是越大越好、过分追求大、追求长远，很可能一事无成。以小见大是不可能全面的。以大见小也不真切。大和小虽有不同，却各有各的合宜之处。如何把握度，则是艺术。

# 读书的意义

- 交志趣相同的朋友
- 与好友读相同的书，找相同的话题
- 审视人生
- 刷短视频、害怕错过某些内容的快节奏其实是没有节奏的，
- 一本好书，可以帮助我们重新认识自己、认清世界，弄清疑惑，并成为一个更好的人。
- 如果为了某个目的寻找书来读，可能未必会达到预期的效果。
- 选书时、看到什么就读什么，选到“坏书”，就扔到一边，也没有浪费什么时间。
- 每个人都可以选择自己的生活方式，但是阅读本身就是一种生活方式。
