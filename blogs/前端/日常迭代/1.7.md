# 1.7

![](2026-01-07-16-25-27.png)

```html
<div class="parent">
  <div class="child"></div>
  <div class="search"></div>
</div>

<style>
  .parent {
    width: 350px;
    postion: relative;
  }

  .child {
    width: 350px;
  }

  .search {
    width: 40px;
    position: absolute;
    right: 0;
    top: 0;
  }
</style>
```

问：search 会遮盖 child， 但是 child 宽度与 parent 相同，如何让 child 可以左右滚动。

答：child 内部后面增加一个 invisible 元素， 宽度等于 search。

```html
<div class="child"></div>
<div class="invisible"></div>
<div class="search"></div>

<style>
  .invisible {
    visibility: hidden;
    width: 40px;
  }
</style>
```

# 如何理解 git rebase 与 git merge

rebase 是 把我的代码合并到别的地方， 被合并地方为基线。
merge 是把别的代码合并到我的分支， 我的分支是基线。
