# 12.28
java springboot
入口文件指定方式：
方式1：通过pom.xml指定
 <configuration>
     <mainClass>com.example.testapi.TestApiApplication</mainClass>
</configuration>

方式2：通过MANIFEST.MF指定
打包后的jar文件中，META-INF/MANIFEST.MF文件包含主类信息：Spring Boot插件会自动生成这个文件。

方式3：命令行指定：
# 运行指定主类
java -cp target/classes com.example.testapi.TestApiApplication

# 运行jar文件（主类已在MANIFEST.MF中指定）
java -jar target/testApi-1.0.0.jar

方式4： 不指定，springboot自动查找@SpringBootApplication注解

spring boot启动流程：
mvn spring-boot:run -> mvn解析pom.xml文件 ->下载依赖 -> 编译 -> 复制资源文件 -> 启动插件 -> 加载 入口文件 -> 执行main方法 -> spring 容器初始化（加载）->启动tomcat->等待http请求
