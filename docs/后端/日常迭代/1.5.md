# 1.5
JpaRepository: Spring Data Jpa 提供的通用数据访问接口，封装了CRUD、分页、排序等常用数据库操作，不用手写sql

使用方法：
1. 项目中引入Spring Data JPA 依赖。
<dependency> <groupId>org.springframework.boot</groupId> <artifactId>spring-boot-starter-data-jpa</artifactId> </dependency>
2. 配置数据库连接  
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/sanguokill_auth?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
    username: root
    password: 你的密码
    driver-class-name: com.mysql.cj.jdbc.Driver
  jpa:
    hibernate:
      ddl-auto: update  # 自动更新表结构（开发环境用，生产环境建议用none）
    show-sql: true      # 打印生成的SQL（方便调试）
    properties:
      hibernate:
        format_sql: true # 格式化SQL输出
